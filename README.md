## Specification
- microcontoler STM32L476
- LCD resistive touch screen TFT 4 Waveshare 13587
## Pinout
 ### Microcontoler   
  ##### SPI/touch screen
 - ALL female goldpins RESERVED
 ##### SPI/SC_CARD
- MISO --> PB_14
- MOSI --> PB_15
- SCK  --> PB_13
- CS --> PB_11

- !help --> show all of avalible commands.
- !display --> The !display function is used to control the display screen.
